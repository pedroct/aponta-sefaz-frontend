# Agent Handbook

Playbooks prontos para agentes AI colaborarem no frontend.

## Available Agents
- [Code Reviewer](./code-reviewer.md) — revisao de qualidade e estilo.
- [Bug Fixer](./bug-fixer.md) — analise e correcoes.
- [Feature Developer](./feature-developer.md) — implementacao de novas features.
- [Refactoring Specialist](./refactoring-specialist.md) — melhorias estruturais.
- [Test Writer](./test-writer.md) — testes unitarios e E2E.
- [Documentation Writer](./documentation-writer.md) — documentacao tecnica.
- [Performance Optimizer](./performance-optimizer.md) — performance.
- [Security Auditor](./security-auditor.md) — seguranca.
- [Frontend Specialist](./frontend-specialist.md) — UI e fluxos React.
- [Devops Specialist](./devops-specialist.md) — CI/CD e deploy.
- [Architect Specialist](./architect-specialist.md) — arquitetura geral.

## How To Use These Playbooks
1. Escolha o playbook alinhado a tarefa.
2. Enriqueca com `docs/AGENTS.md` e `docs/CODEX_RULES.md`.
3. Rode o agente e capture aprendizados nos docs.
